# Skill Template

Use SKILL.md in this directory as the starting point for any new skill.

## Steps

1. Copy this directory and rename it to your skill name (lowercase, hyphens only)
2. Edit SKILL.md - fill in the frontmatter and replace the placeholder instructions
3. Add a references/ subdirectory for any supporting documents your skill links to
4. Add a scripts/ subdirectory if your skill needs executable scripts
5. Open a PR to community/skills/ following the guidelines in CONTRIBUTING.md

The name field in frontmatter must exactly match the folder name.
